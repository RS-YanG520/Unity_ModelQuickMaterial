# 模型快速材质匹配 (ModelQuickMaterial)

Unity 编辑器插件，用于为 FBX 模型按材质名称**批量搜索并匹配**项目中的材质球，支持逐个对应填充。

---

## 功能

- 从 FBX 模型自动读取嵌入材质槽位列表
- 在指定项目文件夹中按**材质名称**搜索匹配的材质球
- 支持**精确匹配**（完全一致）和**模糊匹配**（包含即可）
- 支持递归搜索子文件夹
- 匹配结果逐条预览，可手动拖入材质覆盖
- 一键批量应用到模型的材质重映射
- 支持清除重映射，恢复为默认嵌入材质

---

## 启动方式

| 方式 | 操作 |
|------|------|
| 右键菜单 | 在 Project 窗口中右键点击 FBX → `快速材质匹配` |
| Window 菜单 | `Window → 快速材质匹配` |
| Inspector 按钮 | 选中 FBX 后，Inspector 顶部自动出现绿色按钮 |

---

## 使用流程

1. 在 Project 窗口中**选中一个 FBX 模型**
2. 右键 → `快速材质匹配` 打开窗口
3. 在文件夹树中**选择材质所在的目录**
4. 设置搜索选项：
   - `包含子文件夹` — 是否递归搜索子目录
   - `精确匹配` — 开启后仅匹配名称完全一致的材质
5. 点击 **🔍 搜索匹配**
6. 检查匹配预览，必要时**手动拖入材质**覆盖自动匹配
7. 点击 **应用匹配** 完成

## 安装

1. 将 `ModelQuickMaterial/Editor/` 目录下的两个脚本放入 Unity 项目的任意 `Editor` 文件夹中
2. Unity 会自动编译，无需额外配置
3. 编译完成后右键 FBX 即可看到 `快速材质匹配` 菜单

> **已在本项目中安装**，路径为 `Assets/ModelQuickMaterial/Editor/`

## 兼容性

- Unity 版本：2022.3 LTS 及以上
- 支持的模型格式：FBX、OBJ、DAE、blend、3DS、DXF、glTF/GLB 等
- 渲染管线：内置管线 / URP / HDRP 均可

## 常见问题

**Q: 为什么搜索不到材质？**
A: 检查是否选中了正确的搜索文件夹，以及材质名称是否与 FBX 中的槽位名称一致（注意大小写不影响）。

**Q: 能否匹配部分名称？**
A: 默认模糊匹配模式即可匹配名称包含的情况。如需完全一致，请勾选「精确匹配」。

**Q: 如何恢复模型的默认材质？**
A: 点击「清除匹配」按钮即可清除所有材质重映射，恢复为 FBX 自带的嵌入材质。
